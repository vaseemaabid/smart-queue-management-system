# Smart Queue Management System

A web application for online token booking and queue management.

## Problem Statement

People spend a lot of time waiting in queues at hospitals, banks, and service centers. This project helps users book tokens and track queue status digitally.

## Features

* Online Token Booking
* Queue Tracking
* Token Management
* JWT Authentication
* Protected API Routes

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Security

* JSON Web Token (JWT)

## API Endpoints

### Login

POST /login

### Book Token

POST /book-token

### Get Queue Status

GET /queue

### Call Next Token

PUT /next-token
